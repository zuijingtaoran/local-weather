# local-weather
_IoT nodemcu weather_
**打造一款显示当地天气，时间的小型联网系统**
### 硬件
+ NodeMCU(TB上15元左右)
+ 0.96寸OLED显示屏IIC接口(TB上15元左右)
+ 面包板(TB上0.5元左右)
+ 安卓数据线，1A充电器
